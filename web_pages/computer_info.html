Welcome to PLACEHOLDER OS!

PlaceHolder OS is your place to hold all your files, programs, and more on your computer device.

Most people use PlaceHolder OS with a keyboard, mouse, monitor, and a user account. This system is excellent for public computers like in a library or internet cafe, as most functions are locked down under admin access.

For admins, there are several available commands:

Echoes and pingbacks

hello

Used to check input in the Terminal.

Input: hello Someone

Expected output: Hello, Someone!

ping

Used to check the time it takes for the terminal to respond. You must use your own stopwatch to time it.

Input: ping

Expected output: pong!

pong

Same as above.

Input: pong

Expected output: ping!

Actions

All actions require admin access.

adminaccess [TRUE / FALSE]

Used to sign in to your admin account to make changes to the operating system, files, and programs.

Input: adminaccess TRUE

Expected output: Attempting to set Admin access set to TRUE.
Admin username and password required.
Enter admin username to continue.

Input: adminaccess [ username ]

Expected output: Username accepted.
Enter password for account : [ username ]

Then you'll be asked to enter your password. If entered correctly, you'll have admin access. If entered incorrectly, some older versions of PlaceHolder OS return and encrypted version of the admin password. Make sure your password is encrypted, safe, and not guessable.

checkdisk

Used to check the disk for viruses and harddrive corruption.

Input: checkdisk

Expected output: No issues reported. All systems functioning within normal parameters.

restorefolder

Used to restore a deleted folder from the recycle bin.

Input: restorefolder

Expected output: [1] folder restored.

Bonus functions

cursoreffects [ true, false ]

Like some pizzazz in your life? Turn on cursor effects for a fun treat!

game [ start ]

Used to spend some fun time during the day! Each PlaceHolder OS was coded with a unique game.

Input: game start

Expected output: Game description . . . .

To play the game, make choices by typing game [ choice ]

setdesktopbackground [ url to image ]

Customize your desktop with any image you want, right from the web!
